Eine Vue.js App mit drei Unterkomponenten (ChatHeader, CodeEditor, EntityList) 
für verschiedenen Funktionen eingebaut wurden. Dargestellt werden 4 User mit 
unterschiedlicher Position, die vorläufig mit einem Bot interagieren.

Der Bot versendet automatisch eine Antwort, wenn man den Bot mit "hallo", "hilfe" oder "code" antwortet.
Mit dem CodeEditor ist der User in der Lage, Quellcodes zu schreiben, auszuführen und 
an alle User zu verschicken.

Führt folgende Befehle zum Ausführen der Vue.js App:

Zur Prüfung der sauberen Code: npm run build

Zum Ausführen der Vue.js App: npm run dev
